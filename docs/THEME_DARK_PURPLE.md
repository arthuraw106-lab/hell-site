# تم جدید Hell Manhwa

تم سایت به مشکی و بنفش تیره تغییر کرد.

## رنگ‌های اصلی

- پس‌زمینه: مشکی خیلی تیره `#030308`
- کارت‌ها: بنفش/مشکی `#0b0b18`
- بنفش اصلی: `#4c1d95`
- violet: `#7c3aed`
- highlight: `#a78bfa`
- متن کم‌رنگ: `#a1a1aa`

## فونت فارسی

فونت فارسی به Vazirmatn تغییر کرد.

از CDN زیر استفاده شده:

`https://cdn.jsdelivr.net/gh/rastikerdar/vazirmatn`

## فایل‌های تغییر کرده

- `frontend/tailwind.config.ts`
- `frontend/app/globals.css`
- `frontend/components/effects/animated-background.tsx`
- `frontend/app/layout.tsx`
