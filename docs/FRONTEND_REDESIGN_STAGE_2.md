# مرحله ۲ بازطراحی فرانت Hell Manhwa

## انجام شد

در این مرحله Theme Foundation پروژه ساخته شد.

### فایل‌های تغییر کرده

- `frontend/tailwind.config.ts`
- `frontend/app/globals.css`

### فایل‌های اضافه شده

- `frontend/components/effects/animated-background.tsx`
- `frontend/components/effects/reveal.tsx`
- `frontend/components/ui/pro-button.tsx`

## قابلیت‌های جدید طراحی

- پالت دارک premium
- رنگ‌های violet، red، gold، cyan
- کلاس `glass`
- کلاس `glass-soft`
- کلاس `neo-card`
- کلاس `premium-border`
- کلاس `text-gradient`
- کلاس `hero-grid`
- کلاس `noise`
- کلاس `skeleton`
- کلاس `custom-scrollbar`
- انیمیشن‌های `floaty` و `pulse-glow`
- کامپوننت `AnimatedBackground`
- کامپوننت `Reveal`
- کامپوننت `ProButton`

## نکته

این مرحله هنوز layout اصلی صفحات را عوض نمی‌کند. فقط پایه استایل و کلاس‌ها را آماده می‌کند تا در مراحل بعدی استفاده شوند.
